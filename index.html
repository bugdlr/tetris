How to Play

Clear horizontal lines of blocks by manuvering the falling blocks to fill each row. Earn more points by clearing multiple lines at once. Speed up the rate of falling by soft or hard dropping. Hard dropping earns an extra point for each row skipped. The blocks will progressively fall faster as you achieve higher levels. The game ends when the blocks reach the top of the playing field.

    KEYBOARD CONTROLS

  • right arrow = move piece right
  • left arrow = move piece left
  • up arrow = rotate piece
  • down arrow = soft drop
  • spacebar = hard drop
  • p = pause/unpause the game
